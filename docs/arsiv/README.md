# docs/arsiv/

Bu klasör **pasif** ve **arşivlenmiş** notları içerir. Aktif geliştirme akışında kullanılmaz, ancak ileride referans olabileceği için silinmemiştir.

## İçerik

### `more_env/`
Eski `more-jumpbridge` projesinden taşınmış konfigürasyon örnekleri (`config.json`, `hesap_bildirim.json`) ve eski bir Copilot talimat sürümü. Yeni özellik tasarımında örnek/kıyas amaçlı incelenebilir.

> ⚠️ Buradaki dosyalar **güncel kuralları yansıtmaz**. Aktif talimatlar için `.github/copilot-instructions.md` ve `docs/runbooks/` klasörünü kullanın.
